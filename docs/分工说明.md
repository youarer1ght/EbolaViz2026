## 成员分工

### A — 数据 + 文档

| 模块 | 具体工作 |
|------|---------|

## 协作机制

- 代码仓库：GitHub youarer1ght/EbolaViz2026
- 视图接口：所有视图统一 `init(dom, store, data)` → `{ render, resize, destroy }`
- 状态管理：B 提供 `createStore` + `reducer` + `action creators`，C 的视图中通过 `store.dispatch(action)` 触发状态变更，通过 `store.subscribe(render)` 响应状态更新
- 数据接口：B 提供 `filterCases()` / `summarizeBy*()` 等工具函数，A 的 CSV 数据经 build_real_data.py 生成 JSON 后由前端 fetch 加载
- 调试接口：`window.__store.getState()` / `window.__data`（B 在 main.js 中挂载）
