# EbolaViz2026 — 用户使用指南

> 面向初次使用的用户，按"先总览、再逐视图、最后学联动"的顺序介绍本系统的全部功能。

---

## 一、启动系统

在项目根目录下执行：

```bash
python3 -m http.server 8080
```

浏览器打开 `http://localhost:8082`。看到页面中出现 **5 个视图的网格布局**，且浏览器控制台输出 `✅ EbolaViz2026 ready.`，即启动成功。

> 若端口被占用，换一个即可（如 `8888`、`3000`）。

---

## 二、页面总览

页面从上到下、从左到右分为 5 个区域：

```
┌─────────────────────────────────────────┐
│  标题栏：系统名称 + 播放/暂停/重置按钮       │
├──────────────┬──────────────────────────┤
│ ① 时空热力图  │ ② 时序趋势图               │
├──────────────┴──────────────────────────┤
│ ③ 平行坐标图（多因素关联）                  │
├──────────────┬──────────────────────────┤
│ ④ 政策时间轴  │ ⑤ 详情与统计面板            │
└──────────────┴──────────────────────────┘
```

这 5 个视图共享同一份数据，**你在任何一个视图中的操作都会自动同步到其他 4 个视图**（联动机制）。

---

## 三、全局控制

### 3.1 播放/暂停时间动画

点击标题栏的 **▶ 播放** 按钮，热力图会自动逐日推进，所有视图的数据随之更新。

- 再次点击 **⏸ 暂停** 停止播放。
- 播放速度固定为 **约 0.8 秒/天**（可在 `js/main.js` 中调整）。

### 3.2 重置所有筛选

点击 **↺ 重置** 按钮，清除所有筛选条件（时间范围、选中区域、选中政策、播放状态），恢复到初始全量数据视图。

### 3.3 键盘快捷键

| 键 | 功能 |
|----|------|
| `Space` | 播放/暂停切换 |
| `R` | 重置所有筛选 |

## 四、分视图操作指南

### ① 疫情时空热力图（左上）

本视图展示各省份的累计确诊病例分布，采用**黄→橙→红渐变色**编码。

**操作方法：**

| 操作 | 效果 |
|------|------|
| **点击省份** | 选中该省所有卫生区，右侧出现该省的放大图 + 卫生区散点列表 |
| **在右侧面板点击卫生区散点** | 单独选中/取消该卫生区 |
| **点击右侧"☑ 全选"** | 选中该省全部卫生区 |
| **点击右侧"✕ 清除"** | 取消该省全部卫生区的选中 |
| **拖动/滚轮缩放地图** | 自由缩放和平移（左侧概览图和右侧省份详情均支持） |
| **悬浮省份** | 省份边框加粗（瞬时高亮，不影响筛选） |

**颜色含义：**
- 灰色 = 该省暂未报告病例
- 浅黄→深红 = 累计确诊病例由少到多
- 金色 = 当前被选中的省份

**右侧面板**（点击省份后出现）：
- 上方显示省名、有病例的卫生区数量、已选中数量
- 中间显示该省的放大版 GeoJSON 地图
- 散点标记代表该省内的各个卫生区，点的大小 ≈ 确诊数（对数比例）
- 每个散点旁标注卫生区名称，选中后变金色、加粗

> **提示**：点击某省后，其它 4 个视图会自动筛选到该省数据。再点击右侧面板中的散点可进一步精细到某个卫生区。

---

### ② 时序趋势图（右上）

展示各地区（或卫生区）每日新增病例的折线图。

**操作方法：**

| 操作 | 效果 |
|------|------|
| **拖拽底部滑块**（dataZoom） | 缩小/移动时间范围 → 所有视图同步更新到所选时间段 |
| **在图内框选**（dataZoom inside） | 同上（鼠标在图内拖拽即可） |
| **点击"📊 概览"按钮** | 切换为全部区域合计的单条曲线 |
| **再次点击"📊 细则"** | 恢复为多地区分色曲线 |
| **悬浮某条曲线** | 该曲线加粗（瞬时高亮） |

**颜色含义：**
- 每种颜色代表一个地区（使用 Tableau 10 调色板，最多支持 10 种区分色）

> **提示**：dataZoom 是系统的时间筛选入口，用它刷选时间段后，热力图会只显示该时间段的病例，平行坐标和政策轴也会同步过滤。

---

### ③ 平行坐标图（中部全宽）

展示 5 个维度的多因素关联：**人口密度、医生数量、确诊数、死亡率、床位数**。每条线代表一个卫生区。

**操作方法：**

| 操作 | 效果 |
|------|------|
| **点击某条线** | 选中/取消该卫生区 → 所有视图同步更新 |
| **点击"✕ 清除"按钮** | 清除所有选中区域（按钮仅在有选中时可见） |
| **悬浮某条线** | 该线加粗 + 显示完整数据 tooltip |

**视觉编码：**
- **颜色** = 所属省份（每个省份一种颜色）
- **浓淡** = 死亡率高低（死亡率越高，线条越不透明）
- **粗细** = 选中状态（金色粗线 = 已选中，彩色细线 = 未选中）

> **注意**：平行坐标的"刷选"功能因 ECharts 技术限制（蓝色遮罩不可透明）暂未启用。当前采用点击选择 + 按钮清除作为替代方案，同样支持完整的多视图联动。

---

### ④ 防控政策时间轴（左下）

展示 16 条防控政策事件（封锁、疫苗、援助、监测、卫生响应）。灰色背景曲线为每日总病例趋势。

**操作方法：**

| 操作 | 效果 |
|------|------|
| **点击某个政策标记** | 选中/取消该政策 → 详情面板显示政策信息 |
| **悬浮标记** | 显示该政策的详细信息（标题、类型、描述、来源） |

**颜色含义：**
- 🔴 红色 = 封锁/边境管控
- 🟢 绿色 = 疫苗接种
- 🔵 蓝色 = 国际援助
- 🟠 橙色 = 监测/预警
- 🟣 紫色 = 卫生响应

> **提示**：点击政策节点后，该政策的日期位置会出现竖虚线标记，方便在时间轴上对照。

---

### ⑤ 详情与统计面板（右下）

分为上下两部分：

**上半部分 — 统计卡片：**
- 显示当前筛选条件下的**确诊数、死亡数、病死率(CFR)、含疑似总数、疫区数量**
- 显示当前筛选的**时间范围和区域信息**
- 当选中了具体区域时，额外显示该区域的**人口、密度、城镇化率、医生/床位密度**

**下半部分 — 区域排名条形图：**

| 操作 | 效果 |
|------|------|
| **点击某根柱子** | 选中/取消该卫生区 → 所有视图同步更新 |
| **悬浮柱子** | 显示该区域的详细数据 tooltip |

**颜色含义：**
- 绿→黄→红渐变 = 病死率由低到高
- 金色 = 当前被选中的区域

## 五、多视图联动机制（核心特性）

系统中所有视图通过**集中式状态管理**实现实时联动。你在任意视图中做筛选操作，其它 4 个视图会**自动、同步**更新。

| 你在哪里操作 | 操作方式 | 其它视图的变化 |
|-------------|---------|--------------|
| 热力图 | 点击省份/卫生区 | 时序图仅显示该区域曲线；平行坐标高亮对应线条；政策轴显示该区域政策；详情面板更新统计 |
| 时序图 | dataZoom 刷选时间段 | 热力图仅显示该时段病例；平行坐标过滤到该时段；政策轴高亮时段内政策；详情面板更新统计 |
| 平行坐标 | 点击线条 | 热力图选中对应区域（右侧面板自动打开）；时序图仅显示该区域曲线；详情面板更新统计 |
| 政策轴 | 点击政策标记 | 详情面板显示政策详情；时序图可能出现竖虚线标记 |
| 详情面板 | 点击排名柱子 | 热力图选中对应区域；时序图仅显示该区域曲线；平行坐标高亮对应线条 |
| 标题栏 | 播放动画 | 所有视图按日期逐帧更新 |
| 标题栏 | 重置 | 所有筛选清除，恢复到全量数据视图 |

## 六、典型使用流程

### 流程一：探索"什么时候在什么地方爆发"

1. 打开系统 → 观察热力图，找到颜色最深的省份（疫情中心）
2. 按 **▶ 播放** → 观察疫情从 5 月中旬到 8 月中旬的扩散过程
3. 暂停后，在时序图上用 dataZoom **刷选疫情高峰段**（6 月初）
4. 观察热力图和平行坐标的变化

### 流程二：分析"哪个地区风险最高"

1. 在热力图上**点击颜色最深的省份** → 右侧出现卫生区散点
2. 在散点中找到**最大的标记**（确诊最多的卫生区）→ 点击它
3. 观察详情面板中的统计卡片和排名图
4. 在平行坐标图中观察该区域的 5 个维度指标

### 流程三：评估"防控政策是否有效"

1. 先观察政策轴上的事件标记，找到一条**封锁政策**（红色）
2. 点击该标记 → 在时序图上观察**政策日期前后的病例变化**
3. 点击另一条**疫苗政策**（绿色），对比趋势变化
4. 用 dataZoom 缩小到政策前后 2 周的时间范围，仔细对比

### 流程四：自由探索

1. 在任何视图中操作 → 观察其它 4 个视图的联动反馈
2. 按 **R 重置** → 换一个维度继续探索
3. 使用 Space 快捷键快速播放/暂停

## 七、常见问题

**Q: 页面加载后部分图表显示空白？**
A: 等待 2-3 秒让 ECharts 完成渲染。若持续空白，检查浏览器控制台是否有报错，并确认 `data/` 目录下的 JSON 文件存在。

**Q: 点击省份后右侧面板没有变化？**
A: 确认点击的是左侧概览地图上的省份（不是右侧面板中的散点）。右侧面板仅在选中省份后显示。

**Q: 平行坐标图上的线条太多看不清？**
A: 先在时序图上用 dataZoom 缩小时间范围以减少显示的卫生区数量，或在热力图上点击某个省份来筛选。

**Q: 如何取消所有选中？**
A: 点击标题栏的 **↺ 重置** 按钮，或按键盘 `R` 键。

**Q: 时序图上找不到某个地区？**
A: 点击"📊 概览"切换到汇总模式；或先在热力图上点击该省份筛选后再切换回细则模式。
